# ShoppingTrendsAnalysis
This repo analyses shopping trends among various factors.

The dataset used is https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce/ 

Following analyses or similar have been performed 

- Defined and computed below metrics on the dataset using SQL and MS-Excel
- Categories analysis like ranking by metrics, customer affinity, seasonality
- User segmentation by choices, age, purchasing habits, geolocation
- Seller performance in terms of customer loyalty, ratings, revenue and seasonality
